# **326. Power of Three**

[Power of Three - LeetCode](https://leetcode.com/problems/power-of-three/)

```java
class Solution {
    public boolean isPowerOfThree(int n) {
        if(n <= 0) return false;

       while(n > 1 && (n % 3) == 0 ) 
            n /= 3;

        return (n == 1 || n % 3 == 0)? true : false; 
    }
}
```
